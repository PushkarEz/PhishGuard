# PhishGuard 🔐

## 📌 Overview
PhishGuard is a phishing detection system using a browser extension and backend server to identify malicious links and protect users.

## 🚀 Features
- Detects phishing URLs
- Alerts users in real-time
- Backend processing with Python
- Chrome extension integration

## 🛠 Tech Stack
- Python (Flask)
- JavaScript (Chrome Extension)
- HTML/CSS

## ▶️ How to Run
pip install -r requirements.txt  
python app.py  

## 🌐 Live Demo
🔗 https://phishguard-8qm0.onrender.com/  
> Backend is deployed and running successfully on Render.

## 🧩 Chrome Extension Setup
1. Open Chrome → `chrome://extensions/`  
2. Enable **Developer Mode**  
3. Click **Load unpacked**  
4. Select the `extension/` folder  

✅ Extension will start detecting phishing URLs.
